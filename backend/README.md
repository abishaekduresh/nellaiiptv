# Backend (Slim PHP)

## Overview

The backend of **Nellai IPTV** is a REST API built with **Slim PHP**.

## Prerequisites

- PHP >= 7.4
- Composer

## Setup

```bash
cd backend
composer install   # install PHP dependencies
cp .env.example .env   # configure environment variables
php -S localhost:8080 -t public   # start development server
```

## Running Tests

```bash
./vendor/bin/phpunit   # execute unit tests
```

## Project Structure

```
backend/
├─ app/          # application code (controllers, middleware, etc.)
├─ public/       # entry point (index.php)
├─ src/          # additional source files
├─ tests/        # PHPUnit tests
├─ composer.json
└─ .env          # environment configuration
```
