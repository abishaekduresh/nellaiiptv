# Nellai IPTV

## Overview

Nellai IPTV is a full‑stack video streaming platform.
- **Frontend**: Next.js (React) – located in `frontend/`
- **Backend**: Slim PHP REST API – located in `backend/`

## Getting Started

1. Clone the repository.
2. Install dependencies for both parts (see the README files in `frontend/` and `backend/`).
3. Run the development servers.
4. Open the application in your browser at `http://localhost:3000` (frontend) and `http://localhost:8080` (backend).
